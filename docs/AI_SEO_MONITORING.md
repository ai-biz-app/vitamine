## How to Run the Checks

### Google AI Overviews
1. Search the query on Google
2. Look for the "AI Overview" box at the top
3. Check if any cited source links point to `vitamin.ai-biz.app`
4. Note the page title / URL that appears in the citations

### ChatGPT (with Search)
1. Use ChatGPT with web search enabled (GPT-4 or later)
2. Ask the query
3. Look for numbered citation links below the answer
4. Hover over or click citations to see the source URL

### Perplexity
1. Go to perplexity.ai
2. Enter the query
3. Perplexity always shows inline citations with source URLs
4. Check if any citation links to `vitamin.ai-biz.app`
